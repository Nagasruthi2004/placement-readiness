# Git & Collaboration - Mission Tata 1

## Project Description

This task focused on learning the basic Git and GitHub collaboration workflow. The objective was to create a personal workspace inside the repository, make changes only in the assigned folder, and contribute those changes through a Pull Request.

## Tasks Completed

- Forked the main repository to my GitHub account.
- Cloned the forked repository to my local computer.
- Created a folder using my roll number inside `activities/2026-07-16/`.
- Added a `profile.md` file containing my introduction.
- Committed the changes with a meaningful commit message.
- Pushed the changes to my GitHub repository.
- Opened a Pull Request to the original repository.

## Tools Used

- Git
- GitHub
- Visual Studio Code

## Learning Outcome

This mission helped me understand how developers collaborate on shared repositories without affecting the original project directly. It also gave me practical experience in creating and submitting a Pull Request.
